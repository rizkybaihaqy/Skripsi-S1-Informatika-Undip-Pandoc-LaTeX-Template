Skripdown

Tulis Skripsi di Markdown

Skripdown adalah aplikasi web yang memudahkan konversi dokumen skripsi S1 Informatika Undip dari format Markdown menjadi PDF menggunakan pandoc dan LaTeX. Template LaTeX yang digunakan oleh aplikasi ini dapat ditemukan di github.com/rizkybaihaqy/Skripsi-S1-Informatika-Undip-Pandoc-LaTeX-Template.

  • Bibtex
